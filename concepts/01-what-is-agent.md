# 01 | 什么是 Agent？

## 🎯 学习目标

- 理解 Agent 的基本定义
- 了解 Agent 与普通程序的区别
- 掌握 Agent 的四个核心要素

---

## 📖 概念讲解

### 什么是 Agent？

**Agent（智能体）** 是一种能够**自主感知、自主决策、自主执行**的程序系统。与传统程序不同，Agent 不需要人类一步一步告诉它该做什么，而是能够理解目标后自行规划路径并执行。

### Agent 与普通程序的区别

| 特征 | 普通程序 | Agent |
|------|---------|-------|
| **执行方式** | 按预设流程执行 | 自主规划并动态调整 |
| **输入** | 结构化的指令 | 自然语言描述的目标 |
| **错误处理** | 依赖预设判断 | 能够自我反思并纠正 |
| **交互** | 通常单次完成 | 多轮对话、持续迭代 |

### Agent 的四个核心要素

```
        ┌──────────────────────────────────────┐
        │            Perceive (感知)             │
        │    Receive input, understand context  │
        └─────────────────┬────────────────────┘
                          │
                          ▼
        ┌──────────────────────────────────────┐
        │              Think (思考)              │
        │        Reason using LLM inference      │
        └─────────────────┬────────────────────┘
                          │
                          ▼
        ┌──────────────────────────────────────┐
        │              Act (行动)                │
        │      Call tools, perform operations    │
        └─────────────────┬────────────────────┘
                          │
                          ▼
        ┌──────────────────────────────────────┐
        │           Feedback (反馈)              │
        │    Adjust next step based on result    │
        └──────────────────────────────────────┘
```

### 第一个核心要素：感知（Perceive）

Agent 通过两种方式感知世界：

1. **用户输入** — 直接接收用户的自然语言指令或问题
2. **工具返回** — 通过执行工具后获得的信息来感知环境变化

### 第二个核心要素：思考（Think）

Agent 的"大脑"是大语言模型（LLM），它能够：
- 理解自然语言中的意图
- 将复杂任务分解为可执行的子步骤
- 在执行过程中进行逻辑推理

### 第三个核心要素：行动（Act）

Agent 通过**工具（Tools）** 与世界交互。常见的工具包括：
- 搜索引擎（获取最新信息）
- 文件系统（读写文件）
- 代码执行器（运行代码）
- API 调用（与外部服务通信）

### 第四个核心要素：反馈（Feedback）

Agent 不是一次性执行完所有操作，而是：
1. 执行一个步骤
2. 检查结果
3. 根据结果决定下一步
4. 循环直到完成任务

---

## 💡 生活中的类比

想象你让一个**实习生**帮你完成一个任务：

- 你说："帮我整理一下桌上的文件，按日期分类。"

**普通程序**的做法：
> 严格按指令执行——扫描文件→提取日期→创建文件夹→移动文件。如果遇到一个新情况没有预设处理方式，它就会停下或报错。

**Agent** 的做法：
> 先观察桌上有哪些文件→规划整理步骤→开始分类→遇到不认识的格式会主动问你→根据你的反馈调整→完成并汇报。

关键区别：**Agent 能够理解目标，而不是机械执行指令。**

---

## 📝 章节回顾

**记住这三个关键点：**

1. **Agent ≠ 普通程序**
   普通程序执行固定流程，Agent 理解目标后自主规划路径

2. **四要素循环**
   感知 → 思考 → 行动 → 反馈 →（循环直到完成任务）

3. **工具是 Agent 的"手"**
   没有工具，Agent 只能空想；有了工具，Agent 才能真正改变世界

**核心公式：**

```
Agent = LLM（大脑）+ Tools（手）+ Memory（记忆）+ Planning（规划）
```
