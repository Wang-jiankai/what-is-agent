# 02 | 练习：让 Agent 学会规划

## 🎯 练习目标

- 理解任务分解的思路
- 能判断一个任务适合哪种规划模式
- 能给 Agent 设计合理的执行计划

---

## 📋 练习要求

### 基础任务（必做）

**任务：为"泡一杯绿茶"设计规划步骤**

1. 思考：如果让你教一个机器人泡绿茶，你会给它哪些步骤指令？
2. 把这些步骤按**线性顺序**写出来（标注序号）
3. 标出哪些步骤可以**并行执行**

### 进阶任务（选做）

**任务：判断以下场景适合哪种规划模式**

| 场景 | 你的判断 |
|------|---------|
| 1. 帮用户预订一周的旅行行程 | Plan-and-Execute / ReAct？为什么？ |
| 2. 在陌生城市找一家餐厅 | Plan-and-Execute / ReAct？为什么？ |
| 3. 写一篇 5000 字的技术博客 | Plan-and-Execute / ReAct？为什么？ |

---

## 🔍 思考题

1. 如果一个任务的步骤之间有依赖关系（比如"先注册才能登录"），规划时需要注意什么？
2. 为什么有些任务"计划赶不上变化"？这种任务适合哪种规划模式？
3. Agent 的"规划能力"和人类的"规划能力"，本质区别在哪里？
